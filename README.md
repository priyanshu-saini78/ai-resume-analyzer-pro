# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes using Google Gemini AI and compares them against a job description to provide ATS-style feedback.

## 🚀 Live Demo

https://ai-resume-analyzer-pro-six.vercel.app

---

## ✨ Features

- 📄 Upload Resume
- 🤖 AI Resume Analysis
- 📊 ATS-style Feedback
- 🎯 Resume vs Job Description Matching
- 💡 Resume Improvement Suggestions
- ⚡ Fast Response using Google Gemini AI

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- n8n
- Google Gemini AI
- AI Agent
- REST APIs
- JSON
- Vercel

---

## ⚙️ Architecture

Frontend (HTML, CSS, JavaScript)
        ↓
REST API
        ↓
n8n Workflow
        ↓
AI Agent
        ↓
Google Gemini AI
        ↓
Resume Analysis Response
